PROJECTS

  1. Corona Virus Tracker : Springboot, Thymeleaf, HTML, Bootstrap, Java
    • Dashboard to display Coronavirus cases in different regions along with other insights like increase in cases since the previous day in each area, total patients, etc.
    • Data is derived from a regularly updated GitHub repository and the system is scheduled to retrieve data every day from this repository using cron.

  2. Movie Rating Microservice : Java, Spring Cloud, Eureka server, Springboot
    • 3 Microservices - The first one gets the movie name from the second microservice and its rating from the third microservice.
    • The first microservice then gives the overall data of the given movie - Movie ID, name, and rating.

  3. Criminal psychology analysis : Python, HTML, CSS, Pandas, Numpy, MongoDB, Docker
    • Gets review from the user for any particular scene that depicts a crime in the movie
    • This data is analyzed to grasp how the user felt while watching it
    • This in turn is correlated with a person's emotion when that crime happens in real life.
    • It helps in understanding the criminal's emotions and mindsets
